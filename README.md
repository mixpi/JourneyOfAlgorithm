# JourneyOfAlgorithm
## 项目介绍
目前计划分成两个部分，第一部分纯粹的算法，通过LeetCode或者其他网站刷题，目的是熟练使用常见的算法。在算法熟练之后，开启第二部分，从头开始写代码，不像LeetCode一样只编写代码部分，这么做一方面是熟悉语言，扎实自己的基本功，另一方面也锻炼自己的代码设计能力。
